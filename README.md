# 🛒 Walmart Weekly Sales Prediction System

A Machine Learning powered web application that predicts Walmart weekly sales using economic, seasonal, and store-level data.

---



This project builds an end-to-end Retail Sales Forecasting System:

- 📊 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning Model (Random Forest)
- 🌐 Flask Web Application
- ☁ Deployment Ready

The system predicts weekly sales based on:

- Store ID
- Holiday Flag
- Temperature
- Fuel Price
- CPI (Consumer Price Index)
- Unemployment Rate
- Month

---

## 📈 Model Performance

| Metric | Value |
|--------|--------|
| R² Score | **0.93** |
| MAE | ~72,000 |
| Algorithm | Random Forest Regressor |

The model explains 93% of variance in sales, showing strong predictive capability.

---

## 🔍 Key Insights from EDA

- December has highest average weekly sales.
- Store performance contributes ~66% to prediction importance.
- CPI and Unemployment moderately impact retail demand.
- Official Holiday flag had minimal predictive influence.
- Significant seasonal demand spikes observed during Christmas week.
